# NotepadV2

Like notepad, but better.  

Can compile LaTeX. Has GenAI.  

Please use "pip install -r requirements.txt"

Features :  

1. LaTeX support 
2. PDF compiling
3. AI generated notes
4. AI notes to text
5. Spellcheck
6. Templates
7. Speech to text
8. Audio file transcription  

More to come!
